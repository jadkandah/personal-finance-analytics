# Personal Finance Analytics Platform

An end-to-end personal finance analytics system for tracking, analyzing, and forecasting personal expenses using Python and SQL.

This project focuses on building a complete data pipeline, from data ingestion and cleaning to analysis, visualization, and predictive modeling.
